# wallet_test

  Install dependencies:

```console
$ npm install
```

  Setup MongoDB
```
https://www.mongodb.com/docs/manual/administration/install-community/
```

  Create .env file at root of the project and add below db uri:
```
MONGO_URI="mongodb://127.0.0.1:27017"
```

  Start the server:

```console
$ npm start
```
